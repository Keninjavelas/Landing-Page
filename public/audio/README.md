# Audio Assets

This directory contains all audio files for the Retro-Futuristic Portfolio.

## Required Files

### Page Flip Sound Effect
- `page-flip.mp3` - Short sound effect for page transitions

### Music Playlists

#### Classical (`/classical/`)
Add at least 3 MP3 files named:
- `track1.mp3`
- `track2.mp3`
- `track3.mp3`

#### Jazz (`/jazz/`)
Add at least 3 MP3 files named:
- `track1.mp3`
- `track2.mp3`
- `track3.mp3`

#### Piano (`/piano/`)
Add at least 3 MP3 files named:
- `track1.mp3`
- `track2.mp3`
- `track3.mp3`
